# Password Hasher

This project demonstrates a simple password hasher using Java and the JBCrypt library. 
The project is configured to use Java SDK 17.2.

## Prerequisites

- Java SDK 17.2
- Maven (for dependency management)

## Getting Started

### 1. Clone the Repository
